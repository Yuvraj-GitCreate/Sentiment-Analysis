# Sentiment Analysis App 😊

## 📌 Overview

This project predicts emotions from text using Machine Learning.

## 🚀 Features

* Predicts: sadness, anger, love, joy, fear, surprise
* Built with Logistic Regression + TF-IDF
* Interactive UI using Streamlit

## 🧠 Model Details

* Algorithm: Logistic Regression
* Vectorizer: TF-IDF
* Dataset: CSV with 6 emotion classes

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
